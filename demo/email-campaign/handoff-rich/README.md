# Email launch — strong handoff (demo)

## Goal

Announce Agent Brief workflow improvements to **invited beta workspaces only**. No cold outreach.

## Non-negotiables

- Do not import contacts, change unsubscribe settings, or send live email without named human approval.
- Do not paste raw customer PII into drafts; use segment codes only (`seg-*`).
- Success means: approved copy + segment plan + send window — not "message left the outbox."

## Send window

Prefer Tue–Thu 09:00–11:00 recipient local time (document assumptions if unknown).
